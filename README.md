# Interview Platform - MERN Stack

Real-time technical interview platform with video calling, collaborative code editing, and instant chat.

## Features
- HD Video/Audio Calls (Stream.io)
- Real-time Code Editor (Monaco + Socket.io)
- Multi-language Support (JS, Python, Java)
- Secure Authentication (Clerk)
- Responsive UI (DaisyUI)

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Client
```bash
cd client
npm install
cp .env.example .env
npm run dev
```

## Environment Setup
- Get Clerk keys: https://clerk.com
- Get Stream keys: https://getstream.io
- MongoDB Atlas: https://mongodb.com/atlas
